## User Experience Details

### Success Flow
1. User submits form
2. Button shows loading state
3. On success: modal closes
4. Header updates immediately
5. Success message shown

### Error Flow
1. User submits invalid data
2. Error box appears at top
3. Field highlights (if applicable)
4. Button re-enables
5. User can try again

### Password Reset Flow
1. User enters email
2. Clicks "Forgot Password?"
3. Success message in error box (green)
4. Check email prompt
